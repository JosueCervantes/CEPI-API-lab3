Des maintenant, j'ai ajoute a mon projet la librerie de Express pour gerer
les images et l'envoie des elles vers l'api au dossier media.

Avant, j'avais un code pour transformer les images a base64, mais 
c'etait un json tellement grand pour chaque image, et des fois 
ca ne marchait pas. Donc, j'ai mis express pour envoyer les images a mon 
dossier media et dans l'api on aura seulement l'URL de l'image dans media.

C'est plus facile de gerer les images de cette maniere.

Avec npm i, express sera installe pour telecharger Express

NOTE IMPORTANTE: Pour lancer le serveur le commande est:

---------------------------------
node server.json
----------------------------------

et apres liver server
